# Análise Estatística de Produtos de Supermercado (Chile)

## Objetivo

Este projeto tem como objetivo aplicar conceitos estatísticos e de visualização de dados para analisar preços e descontos em produtos de um supermercado no Chile.

---

## Base de Dados

O dataset contém informações sobre produtos, incluindo:

- Title: Nome do produto  
- Marca: Marca do produto  
- Preco_Normal: Preço sem desconto  
- Preco_Desconto: Preço com desconto  
- Preco_Anterior: Preço anterior ao desconto  
- Desconto: Valor do desconto aplicado  
- Categoria: Categoria do produto  

---

## Tecnologias Utilizadas

- Python  
- pandas  
- matplotlib  
- seaborn  
- plotly
