# Análise Exploratória de Dados em E-commerce

## Descrição

Este projeto realiza uma Análise Exploratória de Dados (EDA) a partir de um conjunto de dados de e-commerce. O objetivo é compreender padrões, identificar relações entre variáveis e extrair insights relevantes sobre preços, avaliações e características dos produtos.

A análise é conduzida utilizando Python e bibliotecas consolidadas de manipulação e visualização de dados.

---

## Tecnologias Utilizadas

- Python
- pandas
- matplotlib
- seaborn

---

## Metodologia

A análise segue as etapas clássicas de um fluxo de ciência de dados:

### 1. Leitura e compreensão dos dados
- Carregamento do dataset
- Inspeção inicial das variáveis
- Verificação de tipos de dados e valores ausentes

### 2. Análise estatística descritiva
- Cálculo de métricas resumo
- Identificação de dispersão e possíveis outliers

### 3. Tratamento e padronização
- Normalização de categorias de temporada
- Padronização de informações relacionadas ao público-alvo

### 3. Visualização de dados
Criação de gráficos para explorar padrões e relações entre variáveis.

---

## Visualizações Implementadas

- Histograma: distribuição dos preços dos produtos  
- Gráfico de dispersão: relação entre preço e nota  
- Mapa de calor: correlação entre variáveis numéricas  
- Gráfico de barras: distribuição de produtos por temporada  
- Gráfico de pizza: proporção de produtos por público-alvo  
- Gráfico de densidade: distribuição das avaliações  
- Gráfico de regressão: relação entre preço e número de avaliações  

---

## Principais Objetivos da Análise

- Avaliar a distribuição de preços e identificar possíveis outliers  
- Investigar a relação entre preço e avaliação dos produtos  
- Identificar correlações entre variáveis relevantes  
- Analisar a segmentação do catálogo por público e temporada  
- Compreender o comportamento das avaliações dos consumidores
