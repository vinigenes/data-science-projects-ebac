# Análise de Vendas com SQL e Python

## Descrição do Projeto

Este projeto tem como objetivo aplicar conceitos fundamentais de **SQL** integrados ao **Python**, utilizando uma base de dados de vendas. A proposta consiste em realizar consultas, agregações e transformações diretamente em um banco de dados SQLite, simulando um ambiente real de análise de dados.

---

## Tecnologias Utilizadas

- Python
- Pandas
- SQLite3
- SQL
