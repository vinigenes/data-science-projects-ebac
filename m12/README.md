# Análise Exploratória - Limite de Crédito

## Objetivo

Este projeto tem como objetivo identificar os principais fatores que influenciam o limite de crédito concedido a indivíduos, por meio de análise exploratória de dados (EDA) e visualizações.

---

## Sobre os Dados

O conjunto de dados contém informações sobre diferentes indivíduos, incluindo:

- Nome  
- Idade  
- Profissão  
- Salário  
- Limite de Crédito  
- Histórico de Inadimplência  
- Estado Civil  
- Imóvel Próprio  

---

## Tecnologias Utilizadas

- Python  
- pandas  
- numpy  
- matplotlib  
- seaborn
